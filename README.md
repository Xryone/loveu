- A sweet Valentine's message inside the letter

- Responsive design

```
📁 project-folder
```

## 🚀 How to Use
1. Clone this repository:
   ```bash
   git clone https://github.com/macedoflp/valentine.git
   ```
2. Open `index.html` in your browser.
3. Click the **buka** button to reveal the message!
